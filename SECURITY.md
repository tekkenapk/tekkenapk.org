# Security Policy

This repository is maintained for **documentation and transparency purposes only**.

## Supported Scope
- This repository does not contain executable code
- No binaries, scripts, or deployable software are provided

## Reporting a Vulnerability
If you believe you have found a security issue related to
the website **https://tekkenapk.org** or this repository,
please report it responsibly.

## Contact
Please report security concerns via email:

**Email:** tekkenapk.org@gmail.com

## Response
We aim to review and respond to legitimate security reports
within **72 hours**.

Thank you for helping keep our resources safe.
